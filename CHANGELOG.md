# 更新日志

## v1.0.4
- 新增：手动添加匹配功能，选中未匹配行可手动标记为匹配，剧名自动存入剧名库
- 新增：列求和功能，可勾选多列计算合计，支持按当前显示/全部行范围计算
- 优化：匹配行在表格中黄色高亮显示

## v1.0.3
- 新增：自动下载更新功能，检测到新版本后可一键下载安装替换重启
- 新增：应用图标（蓝紫渐变 + "剧"字）
- 优化：exe 和安装包均带有应用图标

## v1.0.2
- 新增：检查更新功能，支持自动和手动检查 GitHub 新版本
- 新增：更新日志显示

- 完整的剧名数据管理系统：后台管理、月份管理、数据导入/匹配/导出
- 剧名库管理（添加、删除、批量导入）

- 搜索过滤、列排序、统计合计

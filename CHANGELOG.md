# 更新日志

## v1.1.0
- 新增：单元格点击选中，底部显示选中值
- 新增：Ctrl+C 快捷键复制选中单元格或整行

## v1.0.9
- 修复：在线更新后启动报 "Failed to load Python DLL" 的问题，改进更新脚本等待旧进程退出并清理临时目录

## v1.0.8
- 修复：更新弹窗按钮不管窗口多小都始终显示在底部

## v1.0.7
- 新增：隐藏列功能，可自由勾选要显示/隐藏的列，支持全选/全不选

## v1.0.6
- 修复：更新弹窗按钮被截掉的问题，加大窗口并确保按钮始终可见

## v1.0.5
- 新增：剧名库搜索功能，实时过滤剧名
- 新增：剧名库导出功能，导出为 Excel 文件
- 新增：后台重命名功能
- 新增：月份重命名功能
- 优化：剧名库支持多选删除

## v1.0.4
- 新增：手动添加匹配功能，选中未匹配行可手动标记为匹配，剧名自动存入剧名库
- 新增：列求和功能，可勾选多列计算合计，支持按当前显示/全部行范围计算
- 优化：匹配行在表格中黄色高亮显示

## v1.0.3
- 新增：自动下载更新功能，检测到新版本后可一键下载安装替换重启
- 新增：应用图标（蓝紫渐变 + "剧"字）
- 优化：exe 和安装包均带有应用图标

## v1.0.2
- 新增：检查更新功能，支持自动和手动检查 GitHub 新版本
- 新增：更新日志显示

## v1.0.1
- 首个发布版本
- 完整的剧名数据管理系统：后台管理、月份管理、数据导入/匹配/导出
- 剧名库管理（添加、删除、批量导入）
- Excel 导入导出，匹配行高亮
- 搜索过滤、列排序、统计合计
